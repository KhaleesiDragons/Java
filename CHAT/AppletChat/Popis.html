How works chat?

Chat server is a application that is made up the combination of two-application, server application (which runs on server side) and client application (which runs on client side).

This application is using for chatting in LAN. To start chatting you must be connected with the server after that your message can broadcast to each and every client.

  1. click on server, and run it.
  2. open ClientApplet

  3. enter your login
  4. chat is working, just write something
